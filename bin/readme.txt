This directory is where the executables are put after a "make install".
There should be 2 programs in here:

degrib (degrib.exe)
  The program without graphical support.

tkdegrib (tkdegrib.exe)
  The program with graphical support.

-----

In addition there may be:
tcl83.dll
  MS-Windows dynamic library for Tcl version 8.3

tk83.dll
  MS-Windows dynamic library for Tk version 8.3
