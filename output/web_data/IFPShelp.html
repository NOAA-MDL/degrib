About the Experimental Interactive Graphical Forecasts

AWIPS, the modernized National Weather Service data display system now in operation across the United States, provides forecasters with the best tools to view and understand the past, present, and future state of the atmosphere. Using a variety of data and new computer technology, forecasters are able to integrate satellite and radar imagery, surface and upper air observations, and numerical computer guidance to produce accurate, detailed, and up-to-date forecasts. The Interactive Forecast Preparation System (IFPS) within AWIPS allows forecasters to prepare and display graphical forecasts as effectively as possible.

The displayed images are experimental, graphical representations of text forecasts. You may notice small differences in detail, otherwise the images should provide a reasonable depiction of the expected weather. To view the graphical forecasts, rollover the times listed. Alternatively, press the loop button, to go to a seperate interface which allows the user to loop/step through forecasts for each element by time. To print a single image, click the "Print" button on the "loop" page. Definitions are also available for each element, by clicking on the element name. Additionally a color legend of the weather combinations and abbreviations used for the "weather" element is available. If applicable, the directional icons near the top of the page can be used to navigate to adjacent experimental graphical forecast sites.



NDFD Prototype Page Navigation 
      -  NDFD Home: Provides user with weather element graphics for CONUS
         and sectors/Metro Areas.
         *   Element Maps: A graphical presentation of forecast grids.
         *   Element Loops: An interactive presentation of forecast grids.
         *   Consistency Maps: A graphical presentation of consistencies.

      -  Consistency Statistcs: A statistical representation of overlapping
         WFO gridpoints.
         *   Current:  A current listing of consistencies between WFOs.
         *   Daily:  A daily compilation of consistency statistics.
         *   Weekly:  A weekly compilation of consistency statistics.
         *   Monthly:  A monthly compilation of consistency statistics.

      -  Latest WFO Update: Shows the last time the NDFD server has received
         grids from a WFO.  Additionally, WFOs are highlighted various colors
         based on how recently the data from that WFO has been updated.
         *   Green Highlight:  Less than one hour old (Fresh).
         *   Dark Green Highlight: 1-12 hours old.
         *   Yellow Highlight: 12-24 hours old
         *   Red Highlight: 24+ hours old (Stale).
